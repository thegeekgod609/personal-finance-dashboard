🔗 Google Drive OAuth

First time? Set up your OAuth Client ID (5 min)
  1. Go to Google Cloud Console → create/select a project
  2. Enable the Google Drive API for that project
  3. Credentials → Create OAuth Client ID → type Web application
  4. Under Authorized JavaScript origins, add the URL this dashboard is hosted at
  5. Paste the generated Client ID above and hit Connect
If Connect fails silently here in the Claude preview, host this file at a fixed URL (e.g. GitHub Pages) and register that origin instead — sandboxed previews don't always support OAuth popups.
